## Installation
1. Install npm dependencies.
    ```
    npm install
    ```
2. Install pip dependencies.
    ```
    pip install -r requirements.txt
    ```
3. Start Postgres (TODO: Explain config)
4. Start the Flask backend.
    ```
    python manage.py run
    ```
5. Start the React frontend.
    ```
    npm run serve
    ```
